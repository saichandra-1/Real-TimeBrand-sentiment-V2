# TrendRadar - Zomato Sentiment Analysis Dashboard V2

Real-Time Brand Sentiment & Social Pulse Analyzer using RoBERTa NLP

## Features
- RoBERTa sentiment analysis (cardiffnlp/twitter-roberta-base-sentiment-latest)
- Twitter + Reddit data analysis
- 8 issue categories detection
- Interactive visualizations
- Word clouds by sentiment
- Strategic insights and recommendations
